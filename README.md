# Gwent Codex

## 📌 Sobre o projeto
Aplicação React que consome uma API externa para exibir cartas de Gwent.

## 🚀 Funcionalidades
- Listagem de cartas
- Busca por nome
- Filtro por categoria
- Página de detalhes
- Página por facção
- Rotas dinâmicas

## 🛠 Tecnologias
- React
- React Router
- Axios
- Vite

## ▶️ Como executar
```bash
npm install
npm run dev

🧱 Arquitetura

Usuário → React → Rotas → API → Dados
