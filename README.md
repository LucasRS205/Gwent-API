## Introduction
Welcome to the Gwent API! This API allows you to interact with the Gwent game data, providing endpoints to access various game resources.

## Installation
To install Gwent API, clone the repository and install the necessary dependencies:
```bash
git clone https://github.com/LucasRS205/Gwent-API.git
cd Gwent-API
npm install
```

## Usage
After installation, you can start the server:
```bash
npm start
```

## Endpoints
- `GET /cards` - Retrieve all cards.
- `POST /decks` - Create a new deck.
